# 📋 Student Registration Ledger

A responsive Student Registration Ledger built using **HTML**, **CSS**, and **JavaScript**. This project allows users to enter student details through a registration form and dynamically displays the submitted information in a responsive table without refreshing the page.

---

## 🚀 Features

-  Student Registration Form
-  Client-side form validation
-  Dynamic table row creation using JavaScript
-  Automatic form reset after successful submission
-  Prevents page refresh using `event.preventDefault()`
-  Auto-generated Registration Number
-  Fully Responsive Design


---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📋 Form Fields

The registration form includes the following required fields:

- First Name
- Last Name
- Email
- Address
- Pincode
- Gender
- State
- Country
